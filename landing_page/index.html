Frequently Asked Questions

What is a reverse shell?

A reverse shell is a type of shell in which the target machine initiates a connection back to the attacker's machine. This is commonly used in cyberattacks to bypass firewalls and gain unauthorized access to systems.

Does this tool require root privileges?

Yes, to monitor all processes and terminate malicious ones, this tool needs to run with root privileges. You can run it using sudo or as the root user.

Can I use this on production servers?

Yes, but we recommend running it in dry-run mode (--dry-run) first to ensure it doesn't terminate any legitimate processes in your environment. Once you're confident in its detection capabilities, you can enable automatic termination.

How do I set up email notifications?

Create a .env file in the same directory as the executable with the following variables:

EMAIL_USER=your-email@gmail.com
EMAIL_PASS=your-app-password
EMAIL_RECIPIENT=admin@yourcompany.com

How can I enable LLM-powered detection?

Add your Google Gemini API key to the .env file as GEMINI_API_KEY=your-api-key and run with the --use-llm flag.
